# Homebrew Tap for otel-front

Official Homebrew tap for [otel-front](https://github.com/mesaglio/otel-front) - Lightweight OpenTelemetry viewer for local development.

## Installation

```bash
brew tap mesaglio/otel-front
brew install otel-front
```

## Usage

After installation, run:

```bash
otel-front
```

The web UI will open automatically at `http://localhost:8000`.

## About

This tap is automatically updated by GitHub Actions on each release of otel-front.

For more information, visit the [main repository](https://github.com/mesaglio/otel-front).
